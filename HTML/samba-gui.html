6.7. Herramienta gráfica de administración: SWAT

SWAT es un utilitario de configuración incluido con el paquete Samba. Para poder iniciar esta herramienta, primero deberá editar el archivo /etc/inet.conf y quitar el comentario (es decir, borrar el # que está al principio de la línea) el cual que dice lo siguiente: swat stream tcp nowait.400 root /usr/sbin/swat swat y luego volver a iniciar los servicios inet con el comando /etc/rc.d/init.d/inet restart.

Desde su navegador web favorito (preferentemente uno gráfico), ingrese la URL siguiente: http://localhost:901. Debería aparecer una ventana de autenticación (Figura 6-1) donde se le pedirá un nombre de usuario y una contraseña. La contraseña de root debería ser lo suficientemente segura.
