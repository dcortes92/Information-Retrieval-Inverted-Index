6.2. ¿Qué es capaz de hacer?

  • un servidor SMB proporciona los servicios de compartir archivos e impresoras al estilo de Windows NT o LAN Manager para los clientes SMB tales como Windows 9x, Warp Server, Unix y muchos otros.

  • un servidor de nombres NetBIOS (1001/1002)

  • un cliente SMB del estilo FTP para acceder a los recursos compartidos de las PC (discos e impresoras) para Unix, Netware u otros sistemas operativos.

  • una extensión tar para los clientes, para poder crear copias de respaldo remotas.

  • una herramienta de administración, de línea de comandos, limitada que soporta algunas características de Windows NT que pueden ser usadas sobre Samba, Windows NT Workstation o Server.

  • la implementación de un gateway (pasarela) de sincronización de contraseñas o de un gateway de impresión entre Unix y Windows NT.

  • dar acceso a los archivos de Windows NT desde Unix.

Para más información sobre todo eso, por favor consulte el sitio web de Samba.
