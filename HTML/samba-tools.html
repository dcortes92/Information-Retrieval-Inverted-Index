6.6. Algunas herramientas

6.6.1. smbclient

Esta es una herramienta parecida a ftp que le permite conectarse a una PC que corre Windows.

  • Para consultar la lista de recursos compartidos accesibles en la PC win a través de SMB, ingrese smbclient -L win -N.

  • Ingrese smbclient //win/trabajo para conectarse al recurso compartido trabajo en la PC denominada win. Ud. también puede ingresar lo siguiente: smbclient \\\\win\\trabajo.

  • Para crear un archivador tar del recurso compartido trabajo, ingrese lo siguiente: smbclient //win/work -Tc trabajo.tar.

  • Para imprimir un archivo carta.txt por la impresora mi_impresora conectada a la PC win, ingrese 
    cat letter.txt | smbclient //win/nombre_de_impresora mi_contraseña \
  -N -c "put - darth"

Para más información, ejecute man smbclient.

6.6.2. smbfs

Para poder montar/desmontar algunos directorios de su máquina Windows sobre GNU/Linux, su núcleo necesita soportar el sistema de archivos smbfs (que es el caso de la distribución Linux-Mandrake). Entonces, Ud. puede usar los programas smbmount y/o smbumount. Por ejemplo ingrese lo que sigue: smbmount "\\win\trabajo" -c 'mount /mnt -u 123 -g 456'. Esto montará localmente el recurso compartido trabajo con un UID local de 123 y un GID local de 456.

6.6.3. smbtar

Permite crear archivadores tar remotamente. Lea la página man para más información acerca de sus numerosas opciones.
