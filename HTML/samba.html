6.1. ¿Qué es Samba?

Bueno, Ud. debe pensar, una implementación de una danza exótica para GNU/Linux. ¿Exactamente, qué significa esto? No, no tiene nada que ver con la danza brasileña. Es un servidor para clientes SMB (el protocolo Server Message Block, Servidor de mensajes por bloques) o su sucesor CIFS (Common Internet File System, Sistema común de archivos de Internet).
